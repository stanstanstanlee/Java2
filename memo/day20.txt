라이브러리 (가져다 쓰는것만 알면됨) 라이브러리에 대한 의심 하지 않음. 잘 되어있기 때문에 배포하는거라
 : 코드 묶음 (자바 묶음은 .jar로 되어있음) java archive file
 : 미리 구현한 코드를 제공하는 파일
 : 라이브러리는 분석 하지 않음
 : ★잘 활용 하는 것이 중요 적재적소에 잘 활용할 수 있는가
